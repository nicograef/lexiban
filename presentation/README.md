# Lexiban — Präsentation

Reveal.js-Präsentation für die Vorstellung der Coding-Challenge.

## Setup

```bash
pnpm install
pnpm dev
```

Öffnet einen Vite-Dev-Server mit Hot Reload.

## Steuerung

- **Pfeiltasten** — Navigation (→ nächste Folie, ↓ vertikale Sub-Folie)
- **Esc / O** — Folienübersicht
- **S** — Speaker Notes
- **F** — Fullscreen
